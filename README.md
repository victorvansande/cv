# Victor Van Sande - CV / Portfolio

Een moderne, statische portfolio-website met glassmorphism, animaties en interactieve elementen. Geen build-stap nodig - pure HTML, CSS en JavaScript.

## Pagina's
- `index.html` - landingspagina met hero, kerncijfers en highlights
- `over.html` - over mij, vaardigheden, talen en interesses
- `opleiding.html` - opleidingen (timeline) en certificaten
- `ervaring.html` - relevante ervaring, stages en werkervaring
- `contact.html` - contactgegevens en contactformulier (mailto)

## Lokaal bekijken
Open `index.html` rechtstreeks in je browser, of start een lokale server:

```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1
# → http://localhost:5180/
```

## Bestanden
- `style.css` - design system (kleuren, glass, layout, responsive)
- `main.js` - interacties (cursor glow, scroll reveals, tellers, tilt, mobiel menu)

## Hosten
Push naar GitHub en zet GitHub Pages aan, of sleep de map in Netlify/Vercel. Alles is statisch.
